# Barchi

A [Quickshell](https://quickshell.org/) bar for [Niri](https://github.com/niri-wm/niri).

## Dependencies
- [quickshell-git](https://aur.archlinux.org/packages/quickshell-git): Available on Arch Linux on the AUR. I don't know about other distros, as I only use arch.
- [qml-niri](https://github.com/imiric/qml-niri): a Quickshell plugin for Niri IPC integration. Follow the instructions in their README to build, or if you use Nix it has a nix flake.
