# Barchi

A [Quickshell](https://quickshell.org/) bar for [Niri](https://github.com/niri-wm/niri).

## Dependencies
- [quickshell-git](https://aur.archlinux.org/packages/quickshell-git): Available on Arch Linux on the AUR. On other distros check the [install docs](https://quickshell.org/docs/v0.3.0/guide/install-setup/).
- [qml-niri](https://github.com/imiric/qml-niri): a Quickshell plugin for Niri IPC integration. Follow the instructions on the projects [README](https://github.com/imiric/qml-niri#installation) for how to install it.
